MixpanelWrapper v0.1
====================

Usage
-----

Mixpanel installation:

- Download the latest Mixpanel SDK at this location: https://github.com/mixpanel/mixpanel-unity/releases
- Set your tokens as described here: https://mixpanel.com/help/reference/unity

MixpanelWrapper installation:

- Add this package to your project (if not already done).
- Call MixpanelWrapper.AppLaunched() to track app launch.
- Call MixpanelWrapper.GamePlayed(int score, int gameCount) to track a game played.
